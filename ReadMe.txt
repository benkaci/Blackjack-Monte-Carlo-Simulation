BlackJack Counting Cards_FINAL is a notebook containing the logic for counting cards implementation.
The percent thresh variable is used to assign a probability threshold to for hitting.

All helper functions are included in the notebook in chronoligical order. Running all will in sequential order will produce working code.
The last line will show the game record which includes player actions, percentages, wins and losses.


The BlackJack 16 is the notebook with slighlt modified logic to hit only based on a player total of 16 or less.
The helper functions are exactly the same and this notebook can be run the exact same way.
This code also includes the percent thresh variable however it is not used in any logic.
